## Purpose

Guides are task-oriented documentation. They help users:
- Get started quickly
- Configure the system
- Perform common operations
- Troubleshoot issues

## Guide Types

- **Tutorials**: Step-by-step learning experiences
- **How-To Guides**: Task-focused instructions
- **Reference**: Quick lookup information
- **Explanations**: Conceptual understanding

## Contributing

When writing guides:
1. Start with what the user wants to achieve
2. Provide clear, tested steps
3. Include expected outcomes
4. Add troubleshooting sections for common issues

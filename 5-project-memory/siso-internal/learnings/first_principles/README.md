# First Principles

**Purpose:** First principles analysis and reviews

## Reviews

First principles reviews are conducted periodically to evaluate project direction and decisions.

### Review Schedule

- Every 5 runs: Review last 5 learnings
- Every 10 tasks: Synthesize patterns
- Monthly: Full project retrospective

### Review Template

```markdown
# First Principles Review - YYYY-MM-DD

## What Worked Well

## What Was Harder Than Expected

## What Would We Do Differently

## Patterns Detected

## Proposed Improvements
```

## Usage

Store first principles reviews here. Reviews help identify systemic improvements.

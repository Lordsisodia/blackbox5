# Learnings

**Purpose:** Project insights and lessons learned

## Structure

```
learnings/
├── patterns/           # Extracted patterns
├── first_principles/   # First principles reviews
├── improvements/       # Improvement suggestions
├── retrospectives/     # Retrospective summaries
└── recent.md          # Recent learnings
```

## Usage

Store post-completion insights from tasks and runs. Archive old learnings periodically.

### Patterns
Reusable patterns discovered during development.

### First Principles
First principles analysis and reviews.

### Improvements
Suggestions for process and system improvements.

### Retrospectives
Post-project and post-sprint retrospective summaries.

## Quick Navigation

```bash
# View recent learnings
cat learnings/recent.md

# List all patterns
ls -la learnings/patterns/

# View retrospectives
ls -la learnings/retrospectives/
```

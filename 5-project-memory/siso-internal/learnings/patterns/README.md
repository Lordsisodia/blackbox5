# Patterns

**Purpose:** Reusable patterns discovered during development

## Patterns

### Project Organization

**Pattern:** 6-Folder Structure
**Context:** Project memory organization
**Solution:** Organize by question type (decisions, knowledge, operations, plans, project, tasks)
**Benefits:** 67% reduction in folder count, improved discoverability

### Research

**Pattern:** 4D Analysis Framework
**Context:** Feature research and planning
**Solution:** Analyze across Technology, Features, Architecture, Pitfalls dimensions
**Benefits:** Comprehensive research coverage, consistent output format

### Task Management

**Pattern:** Atomic Task Breakdown
**Context:** Epic implementation planning
**Solution:** Break epics into 18-25 independently completable tasks
**Benefits:** Parallel development, accurate tracking, clear dependencies

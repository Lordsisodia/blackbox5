# Templates

**Purpose:** Templates for consistent documentation

## Subfolders

- `root/` - Root file templates (STATE.yaml, ACTIVE.md, etc.)
- `tasks/` - Task templates
- `goals/` - Goal templates
- `epic/` - Epic templates
- `decisions/` - Decision templates
- `research/` - Research templates
- `operations/` - Operations templates

## Usage

Copy templates to appropriate locations and fill in FILL_ME sections.

## Template Naming

- `*.template` - Raw template files
- `*.md` - Markdown templates
- `*.yaml` - YAML templates

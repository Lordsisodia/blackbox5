# Plans

**Purpose:** Track what we're building

## Subfolders

- `active/` - Active epics and features
- `archived/` - Completed/archived plans
- `briefs/` - Product briefs
- `features/` - Feature management
- `prds/` - Product requirements

## Quick Navigation

```bash
bb5 plan:list
bb5 plan:show [name]
```

## Epic Structure

Each epic folder contains:
- `epic.md` - Epic specification
- `README.md` - Epic overview
- `metadata.yaml` - Estimates, status
- `research/` - Research findings
- `tasks/` - Symlinks to linked tasks

# Goals

**Purpose:** Track all project goals

## Subfolders

- `active/` - Active goal folders (IG-XXX)
- `core/` - Core project goals (CG-XXX)
- `completed/` - Completed goals
- `templates/` - Goal templates

## Quick Navigation

```bash
bb5 goal:list
bb5 goal:show [ID]
```

## Goal Naming

- **IG-XXX**: Improvement Goal
- **CG-XXX**: Core Goal

## File Structure

Each goal folder contains:
- `goal.yaml` - Goal definition
- `timeline.yaml` - Progress tracking
- `journal/` - Daily notes
- `plans/` - Symlinks to linked plans
- `tasks/` - Symlinks to linked tasks

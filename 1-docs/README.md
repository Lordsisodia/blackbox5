## Navigation Guide

### Folder Naming Convention

- **Numbered folders (01-, 02-, etc.)**: Main documentation sections
  - Ordered sequentially for easy navigation
  - Each contains a README.md explaining its contents
  
- **Named folders**: Content within sections or cross-cutting concerns
  - Use descriptive names with hyphens (kebab-case)
  - Group related documentation together

### File Naming Convention

- **README.md**: Entry point for each directory
- **ALL-CAPS.md**: Main documentation files (DESIGN.md, IMPLEMENTATION.md, etc.)
- **kebab-case.md**: Multi-word file names
- **Title-Case.md**: (Legacy) Being standardized to kebab-case

### Finding What You Need

1. **Learning concepts**: Start with [01-theory/](01-theory/)
2. **Understanding implementation**: Visit [02-implementation/](02-implementation/)
3. **How-to guides**: Check [03-guides/](03-guides/)

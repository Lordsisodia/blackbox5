## Core Principles

1. **Trust but Verify** - Always validate assumptions
2. **One Task Per Loop** - Focused execution
3. **Context is King** - Preserve and communicate context
4. **Git is Truth** - Version control as source of reality
5. **Document Everything** - Knowledge preservation
6. **Fail Fast, Learn Faster** - Rapid iteration
7. **Automate the Automatable** - Remove toil

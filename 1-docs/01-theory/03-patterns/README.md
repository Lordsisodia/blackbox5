# Patterns

> Design patterns used in BlackBox5

## Overview

This directory documents recurring design patterns, architectural approaches, and best practices used throughout the BlackBox5 system.

## Key Patterns

- **YAML Communication** - Structured, version-controlled inter-agent messaging
- **Heartbeat Pattern** - Health monitoring and status reporting
- **Queue-Based Work Distribution** - Decoupled task assignment
- **Phase Gates** - Quality checkpoints in workflows
- **Decision Registry** - Audit trail for all decisions

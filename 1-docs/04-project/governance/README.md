# Governance

> Project governance and policies

## Overview

This directory contains governance documentation, including project policies, decision-making processes, and contribution guidelines.

## Files

| File | Purpose |
|------|---------|
| `contributing.md` | How to contribute to the project |
| `code-of-conduct.md` | Community standards |
| `decision-making.md` | How decisions are made |
| `roadmap-process.md` | Roadmap planning process |
| `security-policy.md` | Security disclosure policy |

## Governance Structure

- **Maintainers** - Core project maintainers
- **Contributors** - Active contributors
- **Community** - Users and supporters

## Decision Making

1. **RFC Process** - For major changes
2. **Lazy Consensus** - For minor changes
3. **BDFL** - For contentious decisions

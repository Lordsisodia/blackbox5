# Engine Guides

This directory contains engine-specific guides and the knowledge base for Blackbox5.

## Contents

### core/
Core engine guides and documentation.
- System architecture guides
- Core component documentation
- Internal mechanics

### knowledge/
Knowledge base modules and utilities.
- `catalog.py` - Guide catalog management
- `guide.py` - Guide execution and display
- `recipe.py` - Recipe definitions and management
- `registry.py` - Registry operations
- `executor.py` - Guide execution engine
- `DESIGN.md` - Knowledge system design

### Guides

Engine guides provide:
- System operation procedures
- Troubleshooting workflows
- Maintenance tasks
- Configuration references

## Purpose

Engine guides are for:
- System operators
- Developers working on the engine
- Troubleshooting issues
- Understanding internal mechanics

## Using Engine Guides

1. Import the knowledge module
2. Use the catalog to find relevant guides
3. Execute guides through the executor
4. Reference recipes for common operations

## Contributing

When adding engine guides:
1. Add to the appropriate knowledge module
2. Update the catalog
3. Include runnable code where applicable
4. Document parameters and expected outputs

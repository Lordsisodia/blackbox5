## Advanced Topics

- **Custom Agent Development** - Build specialized agents
- **Protocol Extensions** - Extend YAML protocols
- **Hook System** - Customize execution flow
- **Plugin Architecture** - Add new capabilities
- **Distributed Operation** - Multi-instance setups

## Prerequisites

Before using these guides, you should:
- Complete all getting started guides
- Understand core concepts thoroughly
- Have hands-on experience with basic operations
- Be comfortable with YAML and shell scripting

## Warning

Advanced modifications can:
- Break existing functionality
- Void support agreements
- Introduce security vulnerabilities
- Make upgrades difficult

Always test in a non-production environment first.

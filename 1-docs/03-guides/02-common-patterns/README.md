## Common Workflows

### Task Creation Pattern
```yaml
# 1. Define clear objective
# 2. Set success criteria
# 3. Provide context
# 4. Specify approach
# 5. Document rollback strategy
```

### Agent Collaboration Pattern
```yaml
# 1. Planner creates task
# 2. Executor claims task
# 3. Executor executes
# 4. Verifier validates
# 5. Results committed
```

### Context Management Pattern
```yaml
# 1. Check CURRENT_CONTEXT.md
# 2. Update THOUGHTS.md
# 3. Document decisions
# 4. Preserve learnings
```
